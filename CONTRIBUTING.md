## 🛠️ Comment contribuer

### Option 1 — Pull Request (recommande)

```bash
# 1. Fork le repo sur GitHub
# 2. Clone ton fork
git clone https://github.com/VOTRE-USERNAME/awesome-az-305-francais.git
cd awesome-az-305-francais

# 3. Cree une branche
git checkout -b ma-contribution

# 4. Fais tes modifs
# ... edit files ...

# 5. Commit avec message clair
git add .
git commit -m "Add: nouvelle ressource X"

# 6. Push sur ton fork
git push origin ma-contribution

# 7. Ouvre une Pull Request sur GitHub
```

### Option 2 — Issue

Si tu n'es pas a l'aise avec git, ouvre simplement une **Issue** :

1. Va sur l'onglet "Issues" du repo
2. Click "New Issue"
3. Choisis le template adapte
4. Decris ta suggestion / correction

## ✅ Standards de qualite

### Format markdown

```
✅ Headers avec emoji pour clarte (# 🎯 Section)
✅ Tables pour comparaisons
✅ Mermaid pour decision trees
✅ Code blocks pour snippets
✅ Liens vers sources officielles
✅ Date de mise a jour
```

### Contenu

```
✅ Verifie sur Microsoft Learn officiel
✅ Mention de la date de l'info (services bougent)
✅ Pas de copy-paste direct de docs Microsoft
✅ Pas de claim non sources
✅ Francais professionnel (pas argot)
```

### Code

```
✅ Mermaid bien indente
✅ Markdown valide (rendu test sur GitHub)
✅ Pas de caracteres speciaux unicode bizarres
```

## 🚫 Ce qui sera refuse

```
❌ Copy de Microsoft Official Curriculum (MOC)
```
